# Burp Suite

Burp Suite cheat sheets, split per tool. Start with the overview for setup, then
jump to the tool you need.

- [burp-overview](burp-overview/) — proxy / CA / scope setup and the "Send to…" workflow that ties the tools together
- [burp-repeater](burp-repeater/) — manual request tampering: auth bypass, IDOR, and injection probes by hand
- [burp-intruder](burp-intruder/) — automated fuzzing and the four attack types (Sniper, Battering ram, Pitchfork, Cluster bomb)
- [burp-extensions](burp-extensions/) — key BApp Store extensions: Autorize (authz/IDOR), Param Miner, JWT Editor, Logger++, Turbo Intruder

Each folder contains a `.md` (readable source) and a `.pdf` (the formatted cheat sheet).
